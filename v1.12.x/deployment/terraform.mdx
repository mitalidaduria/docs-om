# Terraform Deployment

The OpenMetadata Terraform module automates the deployment of OpenMetadata and all its dependencies on an existing Kubernetes cluster. It manages the full application stack (server, database, search engine, and orchestrator) through a flexible provisioner system that lets you choose between managed cloud services, in-cluster Helm charts, or existing infrastructure you already operate.
